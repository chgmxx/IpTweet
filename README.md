# Ip Tweet
Ip Tweet - Java Based LAN Mesenger


## Key Features:
  * Peer to peer instant messaging
  * Group Chat
  * Screen Sharing
  * File sharing with peers
  * cross platform
  * Easy to use GUI
  * No server required
  * Track file transfer progress
  * save message logs
  * Lightweight

## Software Requirements:
  Java 1.7 and above


## How to use:
Double click on the jar and you are ready to go.
